Finish the `mergeArray`() method that will accept two int[] of numbers. Return a new merged array

    Examples

        mergeArray([1, 2, 3], [4, 5]) -> [1, 4, 2, 5, 3]
        mergeArray([5, 25, 1], [0, 3, 6]) -> [5, 0, 25, 3, 1, 6]
        mergeArray([8], [4, 10, 9, 2]) -> [8, 4, 10, 9, 2]
